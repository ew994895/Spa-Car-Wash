## Overview

The Promotional System allows you to create, manage, and track special offers and deals across your website. Promotions can be:
- ✅ Scheduled to start and end automatically
- ✅ Displayed in strategic locations optimized for conversion
- ✅ Tracked for click-through analytics
- ✅ Customized for different services (wash, detailing, membership)
- ✅ Run multiple promotions simultaneously
- ✅ Displayed in multiple ways (banner, popup, section, floating button)

## Creating a Promotion

### Step 1: Open Promotion Manager
- Press **`Ctrl + Shift + P`** (anywhere on the site)
- Click **"Create New Promotion"**

### Step 2: Fill Required Fields

#### **Basic Information** (Required)
| Field | Description | Example |
|-------|-------------|---------|
| **Promotion Title** | Main headline | "Spring Detailing Special!" |
| **Description** | Detailed explanation | "$50 off all Premium Detail packages this month" |
| **Discount Amount** | Value/percentage | "$50 OFF" or "20% OFF" |

#### **Optional Information**
| Field | Description | Example |
|-------|-------------|---------|
| **Promo Code** | Customer code | "SPRING2026" |
| **Start Date** | When to go live | "04/15/2026 9:00 AM" |
| **Expiration Date** | When to end | "04/30/2026 5:00 PM" |
| **Image URL** | Banner image | "https://..." |

#### **Call-to-Action Settings**
| Field | Description |
|-------|-------------|
| **Button Text** | What button says (e.g., "Book Now", "Claim Offer") |
| **Button Action** | What happens when clicked (see [Actions](#actions--buttons)) |
| **Link/Section** | Where button goes (if applicable) |

### Step 3: Configure Display Settings

#### **Placement** (Required)
Choose where the promotion button appears:

| Option | Best For | Customer Behavior |
|--------|----------|-------------------|
| **Above Wash Packages** | Wash promotions | High engagement, early in funnel |
| **Above Detailing Services** | Detailing promotions | Premium service focus |
| **Above Membership Section** | Membership/card deals | Recurring revenue target |
| **Floating Bottom Right** | Any promotion | Always visible, non-intrusive |
| **Hero Section Top** | Major announcements | First impression, maximum visibility |
| **In Header** | Urgent/important offers | Always visible |
| **Custom Position** | Advanced use | Manual X/Y percentage placement |

#### **Display Mode** (Required)
| Mode | Description | Best For |
|------|-------------|----------|
| **Section on Page** | Full promotional card in chosen location | Detailed offers, primary method |
| **Banner Only** | Top banner bar only | Quick announcements |
| **Popup Only** | Popup on homepage visit | High-impact promotions |
| **All Methods** | All of the above | Maximum visibility |

#### **Additional Display Options** (Optional)
- ☑️ **Show as Top Banner** - Sticky banner at top of page
- ☑️ **Show Popup on Homepage Visit** - One-time popup when user visits (per session)

### Step 4: Save Promotion
- Click **"Save Promotion"**
- Promotion is created but **INACTIVE** by default
- Toggle it **ACTIVE** when ready to go live

## Placement Strategy

### Strategic Placement Guide

Our placement options are optimized for conversion and customer retention based on behavioral psychology:

#### **1. Above Wash Packages** (High Conversion)
- **When to Use**: Basic wash promotions, membership upgrades
- **Why It Works**: Customers are in decision-making mode, comparing options
- **Best For**: "Buy 10 washes, get 2 free" or "Upgrade to monthly membership"

#### **2. Above Detailing Services** (Premium Focus)
- **When to Use**: Detailing discounts, seasonal detailing offers
- **Why It Works**: Targets customers already interested in premium services
- **Best For**: "$100 off Ultimate Detail" or "Spring Paint Protection Sale"

#### **3. Above Membership Section** (Recurring Revenue)
- **When to Use**: Membership card promotions, loyalty rewards
- **Why It Works**: Reinforces value proposition right before commitment
- **Best For**: "First month 50% off" or "Bonus washes with new cards"

#### **4. Floating Bottom Right** (Always Visible)
- **When to Use**: General promotions, limited-time offers
- **Why It Works**: Non-intrusive, follows user as they scroll
- **Best For**: "Call today for 20% off" or "Weekend special"

#### **5. Hero Section Top** (First Impression)
- **When to Use**: Major announcements, new services
- **Why It Works**: First thing visitors see
- **Best For**: "Now offering ceramic coating" or "Grand reopening special"

#### **6. In Header** (Maximum Visibility)
- **When to Use**: Urgent time-sensitive offers
- **Why It Works**: Always visible, can't be missed
- **Best For**: "Last day! 40% off all services"

## Display Modes

### Primary Display Mode: Section on Page
**Full promotional card with all details**
- Large eye-catching design
- Full description
- Prominent call-to-action
- Shows promo code
- Displays expiration date
- **Best for**: Detailed promotions that need explanation

### Banner Display
**Horizontal bar at top of page**
- Title + discount amount
- Single call-to-action button
- Dismissible by customer
- **Best for**: Quick announcements, simple offers

### Popup Display
**Modal window on homepage visit**
- Appears once per session
- Full promotional details
- Semi-transparent backdrop
- Easy to close (X button)
- Links to full section
- **Best for**: High-impact promotions, new customer offers

### Combining Display Modes
You can enable multiple modes:
- ✅ Primary: Section on page (main promotion detail)
- ✅ Secondary: Top banner (reminder while browsing)
- ✅ Tertiary: Popup (first-time visitor attention)

## Scheduling

### Automatic Start/End Dates

#### Setting Start Date (Future Promotion)
1. Set **Start Date** to future date/time
2. Promotion shows as "Scheduled to start: [date]"
3. Automatically goes live at specified time
4. Appears as **INACTIVE** until start time

#### Setting Expiration Date (Auto-Hide)
1. Set **Expiration Date** to end date/time
2. Promotion automatically hides after expiration
3. No manual action needed
4. Still saved in system (can be reactivated/duplicated)

### Manual Control
- Toggle **ACTIVE/INACTIVE** anytime
- Overrides scheduling
- Use for emergency on/off

### Example Schedule
```
Spring Detailing Sale
Start: 04/01/2026 9:00 AM
End:   04/30/2026 5:00 PM

- Automatically goes live April 1st at 9 AM
- Automatically hides May 1st at 12:00 AM
- Click tracking continues through entire period
```

## Analytics & Tracking

### What's Tracked Automatically

#### Click Counter
- Every time someone clicks the promotion
- Increments by 1 per click
- Visible in admin panel
- Never resets (permanent record)

#### Placement History
- Which placement option was used
- When promotion was created
- Last time it was edited

1. Open Promotion Manager (`Ctrl + Shift + P`)
2. Scroll to promotion list
3. View metrics for each promotion:

## Managing Promotions

### Activating/Deactivating

**To Activate:**
1. Open Promotion Manager
2. Find promotion in list
3. Click eye icon (👁️)
4. Status changes to **ACTIVE** (green)

**To Deactivate:**
1. Click eye-off icon (👁️‍🗨️)
2. Status changes to **INACTIVE** (gray)
3. Promotion hidden from website
4. Click data still saved

### Editing Existing Promotions

1. Open Promotion Manager
2. Find promotion
3. Click **"Edit"** button
4. Modify any field
5. Click **"Save Promotion"**
6. Changes take effect immediately

### Deleting Promotions

⚠️ **WARNING**: This permanently deletes the promotion and all tracking data

1. Open Promotion Manager
2. Find promotion
3. Click **Delete** (🗑️) button
4. Confirm deletion
5. Promotion removed permanently

## Reset Feature

### ⚠️ DANGER: Reset All Promotions

**What It Does:**
- Deletes **ALL** promotions
- Erases **ALL** click tracking data
- Resets system to default state
- **CANNOT BE UNDONE**

**When to Use:**
- Major mistake in setup
- Starting fresh after testing
- Clearing old/expired promotions in bulk

**How to Reset:**
1. Open Promotion Manager
2. Click **"Reset All"** button (top right)
3. Confirmation warning appears
4. Click **"Yes, Reset Everything"**
5. All promotions deleted

**Alternative**: Instead of reset, manually delete individual promotions

## E-Commerce & Online Payments

### Current Payment Capability Status

**❌ CURRENT STATUS**: The website does NOT currently have the ability to process credit/debit card transactions.

### What You Need to Add E-Commerce

To accept online payments for wash cards, memberships, or services, you need to integrate with a payment gateway:

#### **Recommended Payment Gateways**

1. **Stripe** (Most Popular)
   - Easy integration with React
   - Secure payment processing
   - Support for subscriptions (perfect for memberships)
   - Industry-standard security (PCI compliance)
   - Cost: 2.9% + $0.30 per transaction

2. **Square** (Great for Physical Locations)
   - Unified system for in-person and online payments
   - Works with existing Square hardware
   - Customer loyalty programs built-in
   - Cost: 2.9% + $0.30 per online transaction

3. **PayPal** (Widely Trusted)
   - Customers can pay without creating an account
   - Recognized and trusted brand
   - Quick checkout process
   - Cost: 3.49% + $0.49 per transaction

### What E-Commerce Integration Includes

Once integrated, you'll be able to:

✅ **Accept Online Wash Card Purchases**
- Customers can buy prepaid wash cards online
- Automatic promo code application
- Email confirmation with card details

✅ **Process Membership Signups**
- Recurring monthly/annual billing
- Automatic renewal management
- Member dashboard

✅ **Detailing Service Deposits**
- Accept deposits for premium detailing services
- Reduce no-shows
- Secure appointment booking

✅ **Gift Card Sales**
- Digital gift cards
- Custom amounts
- Email delivery to recipients

### Implementation Requirements

To add payment processing, you need:

1. **Payment Gateway Account** (Stripe/Square/PayPal)
2. **Backend Integration** (Supabase with Edge Functions or custom server)
3. **SSL Certificate** (Already included with hosting)
4. **Terms & Privacy Policy** (Legal requirement for payment processing)
5. **PCI Compliance** (Handled by payment gateway)

### Integration Timeline

- **Setup Time**: 3-5 business days
- **Testing Period**: 1-2 weeks
- **Go-Live**: After successful test transactions

### Next Steps to Enable Payments

1. **Choose Payment Gateway** (Recommendation: Stripe for wash cards + subscriptions)
2. **Create Merchant Account** (Register business with payment provider)
3. **Backend Development** (Integrate payment API with booking system)
4. **Testing** (Process test transactions)
5. **Launch** (Go live with real payments)

### Important Security Note

⚠️ **Never store credit card information on your website.** All payment data is handled securely by the payment gateway (Stripe, Square, PayPal). Your website only stores:
- Order/transaction IDs
- Customer names and contact info
- Purchase history (not payment details)

### Demo Mode Status

The current booking system is in **demo mode**, which means:
- ✅ Customers can fill out booking forms
- ✅ Forms show success messages
- ❌ No actual emails are sent
- ❌ No payment processing occurs
- ❌ No backend database storage

To make it fully functional, you need to integrate:
1. Email service (SendGrid, AWS SES, or Resend)
2. Payment gateway (Stripe, Square, or PayPal)
3. Database (Supabase for order/customer management)

## Best Practices

### ✅ DO's

1. **Test Before Going Live**
   - Create promotion as INACTIVE
   - Preview on website
   - Test button actions
   - Activate when confirmed

2. **Use Expiration Dates**
   - Set automatic end dates
   - Prevents outdated offers
   - Professional appearance

3. **Match Placement to Offer**
   - Detailing promos → Above Detailing
   - Wash cards → Above Membership
   - General → Floating button

4. **Keep Discount Clear**
   - "$50 OFF" is better than "Special discount"
   - "20% OFF" is better than "Big savings"

5. **Track Performance**
   - Check click counts weekly
   - Compare different placements
   - Adjust strategy based on data

6. **Limit Active Promotions**
   - 1-2 active promotions maximum
   - Too many creates confusion
   - Focus on best offer

### ❌ DON'Ts

1. **Don't Leave Expired Promotions Active**
   - Use expiration dates
   - Manual deactivation if needed

2. **Don't Use Vague Descriptions**
   - Be specific about discount
   - Clear terms and conditions

3. **Don't Overuse Popups**
   - Only for major promotions
   - Can annoy repeat visitors

4. **Don't Forget to Update**
   - Seasonal offers need changes
   - Keep content fresh

5. **Don't Delete Without Checking**
   - Lost click data cannot be recovered
   - Edit instead when possible

## Troubleshooting

### Promotion Not Showing Up?
- ✅ Check if **ACTIVE** (green eye icon)
- ✅ Check **Start Date** (hasn't arrived yet?)
- ✅ Check **Expiration Date** (already passed?)
- ✅ Refresh page (`Ctrl + R` or `F5`)

### Banner Dismissed by Customer?
- One-time per browser
- Cleared when customer clears cookies
- This is normal behavior
- Not a bug

### Popup Not Showing?
- Only shows once per session
- Check if **"Show Popup"** is enabled
- Refresh page or open in private/incognito window

### Click Count Not Increasing?
- Counts actual button clicks only
- Not pageviews
- Working as intended
