### **6. ONLINE BOOKING SYSTEM**
**What it does**: Allows customers to request detailing appointments online.

**Location**: 
- Detailing Services section (main booking form)
- Individual detailing package pages (pre-filled with package)

**Booking Process**:
1. Customer selects service/package
2. Fills out form (name, email, phone, vehicle info, preferred date)
3. Adds notes/special requests
4. Clicks "Request Appointment"
5. Sees confirmation message

**Current Status**: Demo mode (form submissions don't send emails yet - requires backend integration)

**Employee Training**: When customer asks "Can I book online?" → Guide them to detailing section, help them fill out form, explain that we'll call them back to confirm.

## 🎯 EMPLOYEE TRAINING QUICK TIPS

### **When Assisting Customers on Website**:

1. **Always start at the top** - Most info visible from homepage
2. **Use scroll navigation** - No clicking required for main sections
3. **Help section is your friend** - All contact info in one place
4. **Chat for after-hours** - Direct closed-time inquiries there
5. **Booking = Detailing section** - Tunnel washes are walk-in only

### **Common Misconceptions to Correct**:

❌ "Can I buy a wash card online?" → Not yet, call to purchase  
❌ "Can I book a tunnel wash?" → No, tunnel is first-come-first-served  
❌ "Is the booking form confirmed?" → No, it's a request - we call back  
❌ "Why didn't I get an email confirmation?" → System in demo mode currently

### **Upsell Opportunities**:

- Customer asking about basic wash? → Mention Elite Ultimate package
- Customer booking exterior detail? → Suggest Signature Complete (inside too)
- One-time detailing customer? → Mention membership savings
- First-time visitor? → Check for active promotions (guide them to it)
