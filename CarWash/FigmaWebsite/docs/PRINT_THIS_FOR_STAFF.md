## 📅 DAILY ROUTINE

### ☀️ MORNING (Opening - 8:00 AM / 9:00 AM)

1. **Open website** on front desk computer
   - Go to: `spacarwash.com` (or `localhost:5173`)

2. **Set Status to OPEN**
   ```
   Press: Ctrl+Shift+S
   Select: OPEN
   Click: Save
   ```

3. **Set Current Wait Times**
   ```
   Press: Ctrl+Shift+X
   Adjust sliders for each service:
   - Full Wash: _____ minutes
   - Hand Wash: _____ minutes
   Click: Save
   ```

---

### 🌅 THROUGHOUT THE DAY

**Update Wait Times as Needed:**
```
Press: Ctrl+Shift+X
Move sliders to current wait times
Click: Save
```

**Typical Wait Times:**
- **Low Traffic** (Green): 15-20 minutes
- **Medium Traffic** (Yellow): 25-35 minutes  
- **High Traffic** (Orange/Red): 40-60 minutes

## 🆘 TROUBLESHOOTING

### Problem: Shortcuts Don't Work

**Solutions:**
1. ✅ Click anywhere on webpage first
2. ✅ Make sure you're on the website page (not another tab)
3. ✅ Press **Ctrl** key (not Cmd)
4. ✅ Hold all 3 keys at same time: **Ctrl+Shift+S**
5. ✅ Refresh page (F5)
6. ✅ Try different browser

### Problem: Changes Don't Appear

**Solutions:**
1. ✅ Make sure you clicked **Save** button
2. ✅ Refresh the webpage (F5)
3. ✅ Clear browser cache (Ctrl+Shift+Delete)
4. ✅ Check you're not in "Private" or "Incognito" mode

### Problem: Forgot How to Open Panel

**Quick Memory:**
- **S** = Status
- **P** = Promotions  
- **X** = eXtra services (wait times)

All use: **Ctrl+Shift+[Letter]**
