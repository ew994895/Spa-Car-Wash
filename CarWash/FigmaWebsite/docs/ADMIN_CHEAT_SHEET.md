## 🎯 ADMIN PANELS QUICK ACCESS

### **1. PROMOTION MANAGER** (Ctrl + Shift + P)
**What it does**: Create and manage special deals/promotions

**Key Actions**:
- Click "Create New Promotion" → Fill form → Save → Toggle ACTIVE
- Edit: Click "Edit" button on existing promotion
- Deactivate: Click eye icon (👁️ → 👁️‍🗨️)
- Delete: Click trash icon (🗑️)
- **RESET ALL**: Top-right "Reset All" button (⚠️ deletes everything)

**Required Fields**:
- Title, Description, Discount Amount
- Placement (where it appears)
- Display Mode (how it shows)

**Optional Fields**:
- Promo Code, Start Date, Expiration Date, Image URL, Button Action

---

### **2. BUSINESS STATUS MANAGER** (Ctrl + Shift + S)
**What it does**: Update open/closed status & hours

**Key Actions**:
- Click status badge in header OR press Ctrl + Shift + S
- Toggle "Currently Open" / "Currently Closed"
- Set reason (weather, holiday, etc.)
- Auto-updates based on schedule you set

## ⚠️ CRITICAL ADMIN MISTAKES TO AVOID

### **PROMOTION SYSTEM**
❌ **DON'T** click "Reset All" unless you want to delete EVERYTHING  
❌ **DON'T** leave expired promotions active (set expiration dates)  
❌ **DON'T** run too many promotions at once (1-2 max recommended)  
❌ **DON'T** delete promotions without checking click data (it's permanent)

**If you make a mistake**:
1. Promotion showing wrong info? → Edit promotion → Fix → Save
2. Promotion won't show? → Check if ACTIVE, check dates, refresh page
3. Deleted promotion by accident? → Can't recover (no undo)
4. Too many test promotions? → Use "Reset All" to clear and start fresh

### **STATUS SYSTEM**
❌ **DON'T** forget to update status during weather closures  
❌ **DON'T** leave "Closed" status on when reopening  

**If you make a mistake**:
1. Wrong hours showing? → Open Status Manager → Update schedule
2. Status stuck? → Refresh page (F5)
3. Emergency closure? → Click badge → Toggle "Currently Closed" → Set reason

## 🔐 ADMIN ACCESS SECURITY

### **Who Has Access?**
- Admin controls available to anyone with website access
- No password required (keyboard shortcuts work for everyone)
- **Important**: Only share admin shortcuts with trusted staff

### **Data Storage**
- All settings stored in browser localStorage
- Changes saved immediately
- Data persists until manually cleared
- Each browser/device has separate settings

### **Recommended Practice**:
- Document all major changes (keep a log)
- Test promotions in INACTIVE mode first
- Don't use personal devices for admin access
- Clear browser cache only when necessary
